# JavaScript фреймворки и библиотеки

## Определения

- **JavaScript фреймворк** - это каркас приложения, написанный на JavaScript. Он отличается от библиотеки JavaScript своим потоком управления. Библиотека предлагает функции, вызываемые ее родительским кодом, в то время как инфраструктура определяет весь дизайн приложения. Разработчик не вызывает фреймворк; вместо этого платформа вызывает и использует код определенным образом. Примеры включают React и Vue.js.
